# Project: MRIO Data Browser

## Goal
Build an online web app to browse Multi-Regional Input-Output (MRIO) data interactively.

## Planned Features
- Load and display MRIO matrices (Z, A, L, Y, etc.)
- Filter by region and sector
- Visualize trade flows
- Explore supply chain linkages
